HTML Text Formatting

HTML contains several elements for defining text with a special meaning. Example

This Text is Bold

This text is italic

This is subscript and superscript

HTML Formatting Elements

Formatting elements were designed to display special types of text:

  • (b) - Bold Text
  • (strong) - Important text
  • (i) - Italic Text
  • (em) - Emphasized Text
  • (mark) - Mraked text
  • (small) - Smaller Text
  • (del) - Deleted Text
  • (ins) - Inserted Text
  • (sub) - subscript Text
  • (sup) - superscript
